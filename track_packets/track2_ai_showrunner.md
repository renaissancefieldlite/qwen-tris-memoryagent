# Track 2 Packet - AI Showrunner

## Submission Thesis

Qwen Showrunner converts an approved source packet into a complete short
demo/drama production packet: script, beat sheet, storyboard, shot list,
generation prompts, asset ledger, and edit decision list.

## Existing Overlap

- Quadro demo-script and source-backed packet style.
- Golden Field Lite narrative continuity.
- RFL deck/pitch public-safe artifact discipline.

## Build Plan

1. Define a public-safe showrunner source packet schema.
2. Build agents for writer, continuity editor, storyboarder, visual prompt
   planner, and compliance/release checker.
3. Add Qwen Cloud calls for script/storyboard planning.
4. Add optional Wan/HappyHorse generation hooks only after credential route is
   verified.
5. Produce a short demo from one cleared public-safe packet.

## Real Evidence Gate

Use only approved public-safe content. The generated story can be creative, but
the source packet and release ledger must be real.

Metrics:

- source facts preserved
- scene continuity pass/fail
- token budget per production stage
- asset ledger completeness
- release-check pass/fail

## Do Not Copy

- private operator-state material
- raw captures
- claim-sensitive mechanics
- anything that turns private continuity into public proof copy
